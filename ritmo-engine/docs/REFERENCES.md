# 참고 논문 & 기술

## Beat/Downbeat Detection
- **Beat This!** (2024) — Efficient beat/downbeat tracking, knowledge distillation 가능
- **Madmom** (Böck et al., 2016) — RNN + Viterbi DBN, 현재 사용 중
- **TCN Beat Tracker** (Matthew Davies et al.) — Temporal Convolutional Network
- **BeatNet** (Heydari & Cwitkowitz, 2021) — Joint beat/downbeat/meter

## Source Separation
- **Demucs** (Meta, 2023) — Hybrid Transformer, 퍼커션 분리 최강
- **HPSS** (librosa) — Harmonic-Percussive Source Separation, 현재 사용 중 (단순)

## Music Structure Analysis
- **MSAF** (Nieto & Bello, 2016) — Music Structure Analysis Framework
- **Spectral Clustering** (McFee & Ellis, 2014) — 라플라시안 세그멘테이션
- **All-In-One** (Taejun Kim, 2023) — 비트+다운비트+구조 통합 모델

## Latin Music Specific
- **Salsa Dataset** (CCRMA) — 124곡 살사 비트 어노테이션
- **Clave Pattern Detection** — 손 클라베(2-3, 3-2) 자동 감지 연구
- **Onset Detection for Percussion** — 콩가/봉고/팀발레스 특화

계정 삭제 안내

최종 업데이트: 2026년 3월 27일

Ritmo 앱 내에서 직접 계정을 삭제할 수 있습니다.

앱 내 삭제 방법

  1. Ritmo 앱을 열고 설정 탭으로 이동합니다.
  2. 화면 하단의 "계정 삭제" 버튼을 탭합니다.
  3. 확인 팝업에서 "삭제"를 선택합니다.
  4. 계정과 모든 관련 데이터가 즉시 삭제됩니다.

삭제되는 데이터

  • 계정 정보 (이메일, 프로필)
  • 클라우드 라이브러리 (등록된 곡 목록)
  • 에디션 데이터 (프레이즈 편집, 포메이션)
  • 커뮤니티 활동 (게시글, 댓글)

주의: 계정 삭제는 되돌릴 수 없습니다. 삭제된 데이터는 복구할 수 없습니다.

이메일로 삭제 요청

앱에 접근할 수 없는 경우, 아래 이메일로 계정 삭제를 요청할 수 있습니다. 가입 시 사용한 이메일 주소를 포함해 주세요.

📧 support@ritmo.kr

요청 후 영업일 기준 3일 이내에 처리됩니다.

Account Deletion

Last updated: March 27, 2026

You can delete your account directly within the Ritmo app.

How to Delete in the App

  1. Open the Ritmo app and go to the Settings tab.
  2. Tap the "Delete Account" button at the bottom of the screen.
  3. Select "Delete" in the confirmation popup.
  4. Your account and all associated data will be deleted immediately.

Data That Will Be Deleted

  • Account information (email, profile)
  • Cloud library (registered track list)
  • Edition data (phrase edits, formations)
  • Community activity (posts, comments)

Warning: Account deletion is irreversible. Deleted data cannot be recovered.

Request Deletion via Email

If you cannot access the app, you may request account deletion by email. Please include the email address used to sign up.

📧 support@ritmo.kr

Requests will be processed within 3 business days.

# Musicality - MVP Milestone Checklist

- [x] Expo 프로젝트 초기화 (SDK 54, TypeScript)

- [x] 다크 테마 기본 적용
- [x] 탭 네비게이션 (Library / Player / Settings)
- [x] 파일 임포트 (mp3, wav, flac, m4a)
- [x] 트랙 리스트 표시 + 삭제
- [x] 오디오 재생 (Play / Pause)
- [x] SeekBar (터치 + 드래그)
- [x] 현재 시간 / 총 시간 표시
- [x] 재생 속도 조절 (0.5x ~ 1.5x)

## 2. 고정 IP 서버 세팅 (오전)

### 네트워크
- 고정IP 설치 완료: 112.169.106.66 (enp5s0)
- 게이트웨이: 112.169.106.65, 서브넷: 255.255.255.192
- NIC 듀얼 구성: enp4s0(가정용 DHCP) + enp5s0(고정IP)

### Policy Routing
- 문제: 기본 게이트웨이가 가정용(192.168.0.1) → 고정IP로 들어온 요청의 응답이 가정용으로 나감
- 해결: `ip rule add from 112.169.106.66 table fixedip` + 전용 라우팅 테이블
- netplan 영구 설정 적용

### nginx + SSL
- nginx 1.24 설치 + api.ritmo.kr reverse proxy 설정
- Let's Encrypt SSL 인증서 발급 (만료 2026-06-16, 자동갱신)
- client_max_body_size 500M, proxy_read_timeout 300s
- Tailscale Funnel 비활성화 (443 포트 충돌 해소)

### 앱 설정
- config.ts: API_BASE_URL → `https://api.ritmo.kr` (고정IP 직접 연결)

### 향후 계획
- ritmo.kr 홈페이지: Vercel 무료 유지
- Vultr VPS: hanjahanja/timeglobe 운영 중이라 당분간 유지

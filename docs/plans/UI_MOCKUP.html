v5 변경사항 — 롱프레스 모드 토글

🎯모드 전환 = [🔢 | 👥] 세그먼트 하나 — 탭=View, 롱프레스=Edit. 색상으로 현재 상태 구분 (teal=view, purple=edit)
🗑️✏️ 버튼 제거 — 별도 edit 토글 불필요. 롱프레스가 edit 진입을 겸함
⚙️기존 v4 유지 — ⚙️ 설정 모달, [S] 슬롯, ↩ undo (edit only), 자동저장
📏컨트롤바 더 심플 — 모드 세그먼트 + 스피드 + 재생 + 볼륨 + undo
